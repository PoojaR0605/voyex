## Overview

VOYEX is a modern travel planning web application that helps users discover destinations, browse curated travel packages, and manage their journeys through a clean, responsive interface.

---

## Features

- 🔐 **User Authentication** — Login and signup flows
- 🧭 **Trip Planning** — Easily plan and organize trips
- 📦 **Travel Packages** — Browse curated travel deals
- 🧳 **Trip Management** — Track and manage upcoming journeys
- 📱 **Responsive UI** — Seamless experience across devices
- ⚡ **Fast Performance** — Powered by Vite

## Getting Started

```bash
# Clone the repository
git clone https://github.com/PoojaR0605/voyex.git
cd voyex

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
